

User Story :
Как Продавец, я хочу публиковать карточки товара с указанием цены, названия, описания и фотографии, наличие, чтобы потенциальные покупатели могли увидеть мой товар и заказать его:
Критерии Приема :
AC 1: На форме создания товара должны быть поля: Наименование товара, цена товара, описание товара, фотография товара, количество товара в наличии
AC 2: Поле цена может принимать только положительные значения.
AC 3: После нажатия кнопки опубликовать, статус карточки переходит в статус "На модерации"
AC 4: Продавец видит свои карточки во вкладке "Мои товары" со статусом "На модерации"
AC 5: После прохождения этапа "На модерации" перевести карточку в статус "Опубликовано"
AC 6: Продавец видит свои карточки во вкладке "Мои товары" со статусом "Опубликовано"
 

Use Case:
UC 1 Публикация карточки товара:
Актор: "Авторизованный Продавец" 
Предусловие : Авторизованный продавец вошел в Личный кабинет и открыл форму добавления товара:

Основной сценарий:
1)Продавец заполняет Обязательные поля в карточку товара: (наименование, цену, описание, количество товара) и Фотографию товара
2)Продавец нажимает кнопку "Опубликовать"
3)Система валидирует введенные данные (Проверка на заполненность, цена>0)
4)Система сохраняет карточку товара в БД со статусом "На модерации/Pending moderation"
5)Система выводит сообщение об успешной регистрации карточки товара и возвращает Продавца во вкладку "Мои товары"

Сценарии исключения:
1)Поля не заполнены: Система подсвечивает обязательные поля красным и блокирует Кнопку опубликовать
2)Неверный формат цены: Система выводит ошибку "Цена должна быть больше 0"
3)Неверный формат фото: Система выводит ошибку в случае если пользователь прикладывает некорректный формат "Приложите фото корректного формата JPG, PNG,JPEG}"
